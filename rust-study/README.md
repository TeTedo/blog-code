## 사전 환경설정(macos)

rust 설치

```
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

```
rustc --version
```

c compiler 설치

```
xcode-select --install
```

cargo 를 주로 사용해서 정리할거 같긴한데 간단한건 그냥 rustc로 컴파일해서 실행하려고 한다.
